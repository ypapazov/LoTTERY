Step 1 — Generate Local Seed

Generate a cryptographically random seed using the Web Crypto API. A PBKDF2 commitment will be computed and displayed.

Step 2 — Fetch Remote Seed

Fetch a random value from random.org, or enter one manually if random.org is unreachable.

Step 3 — Lock Parameters

Set the minimum and maximum values for the output range. These will be visible to the audience.

Step 4 — Human Input

A designated person speaks a random number aloud. Enter it here after the audience has heard it.

Step 5 — Reveal & Verify Seeds

The local and remote seeds are now revealed. The application verifies they match the earlier commitments.

Step 6 — Draw Numbers

Each draw uses the next output from the CSRNG seeded by the XOR of all three sources.
